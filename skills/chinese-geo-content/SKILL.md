# chinese-geo · 内容工程（改写成可被引用的形态）

同样的事实，换种写法，AI 可见度可能明显不同（Princeton 等团队的 GEO 论文在**模拟引擎**实验里，加统计 / 引文 / 出处约 +30–40%；真实引擎未必同幅）。核心：像在直接回答问题、有据可查、读着顺。

## 改写手法 → AI 可见度提升

数字源自 GEO 论文的**模拟引擎**实验（非真实 AI 搜索），方向可信、数值仅参考，按区间说，别承诺确数。

| 手法 | 提升 | 怎么做 |
|---|---|---|
| 加引用来源 | +27~115% | 关键结论后跟"据 X 报告 / 官方文档" |
| 加统计数字 | +33~37% | 把"很多""大幅"换成具体数字 + 出处 |
| 加专家引文 | +28~41% | 引一句权威人物 / 机构的话 |
| 结论前置 | +25% | 段首先给答案，再解释 |
| 提升流畅度 | +15~30% | 短句、去口水、逻辑连接清楚 |
| 权威语气 | +16~25% | 确定性表述（别堆"可能大概也许"） |
| 关键词堆砌 | −10%（有害） | 别做，反伤可见度 |

## 改写清单（判断层）

- **TL;DR 摘要**：长文开头给 2–3 句结论摘要，AI 最爱抽这段。
- **问题式标题**：标题 / 小标题写成用户会搜的问句。
- **答案前置**：每段第一句就是答案（倒金字塔）。
- **数字带出处**：每个数字配来源，AI 更敢引、用户更信。
- **新鲜度**：标注更新日期、给最新数据——AI 联网偏好新内容。
- **中文友好**：正文按字数不按词数（≥300 字），别被"中文没空格"坑到。

## 诚实边界

- 纯文案手法随竞争趋于零和（C-SEO Bench）：大家都加数字时，单靠文案的边际收益下降——别过度承诺。确定性赢面在爬虫准入（chinese-geo-audit）与结构（chinese-geo-structure）。
- 数值是研究估计，按区间表述。
